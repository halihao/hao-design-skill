# JoJoMath 宣传视觉规范

这份文档用于海报、社媒图、活动图、广告概念、落地页首屏视觉和产品宣传截图。当前是骨架，后续需要补充官方认可案例。

## 传播层级

每张宣传视觉都要按顺序回答这些问题：

1. **这是给谁看的？** 学生、家长、年级段、市场或活动目标人群。
2. **具体收益是什么？** 信心、练习效率、掌握度、诊断清晰度、进步或优惠。
3. **为什么可信？** 产品截图、学习机制、报告证据、用户证明、数据点或可信项目细节。
4. **用户下一步做什么？** 开始、扫码、报名、预约、分享、参加或了解更多。

## 视觉方向

- 能使用真实产品 UI、数学内容、进度/报告信号或官方品牌资产时，优先使用真实素材。
- 保持数学学习相关性可见；不要做成泛教育海报，不要只放无关文具或抽象图形。
- 让一个信息成为主角，次级信息只能辅助，不要抢层级。
- 给平台裁切和移动端阅读留足安全空间。
- 待补充：补充官方认可的图片处理方式、角色/吉祥物用法、背景风格和活动案例。

## 版式模式

- **产品证据型**：标题 + 产品截图 + 简短收益 + 行动按钮。
- **家长信任型**：结果主张 + 报告/进度证据 + 安心说明 + 行动按钮。
- **学生激励型**：友好挑战 + 奖励/进度提示 + 立即行动。
- **季节活动型**：时令主题 + 优惠/项目细节 + 产品证据 + 行动按钮。
- **社媒方图型**：一个标题、一个视觉主锚点、一个行动按钮，辅助文案尽量少。

## 文案语气

- 先讲具体学习结果或产品场景。
- 优先使用具体主张，不要泛泛夸张。
- 标题要足够短，适合小屏查看。
- 家长端文案要可信、冷静。
- 学生端文案可以更有能量，但不能过度承诺。
- 待补充：补充官方认可话术库、禁用话术和本地化文案规则。

## 最终设计前需要的信息

- 目标受众和投放位置。
- 格式和尺寸。
- 主推优惠或活动目标。
- 必须包含的品牌标志、产品截图、二维码、行动按钮或法律文字。
- 已确认可以使用的宣传主张，以及特定市场文案。

如果缺少必要信息，只能在明确标注假设的前提下先做方向稿。
