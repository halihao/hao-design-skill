# JoJoMath 案例库

这份文档是设计证据库。后续请持续加入真实截图、Figma 导出图和宣传物料；每个案例都要说明它证明了什么规则。

## 如何添加案例

1. 把素材放入对应目录：
   - `assets/screenshots/`：产品 UI 截图。
   - `assets/figma-exports/`：Figma 导出页面。
   - `assets/marketing-examples/`：已确认可以参考的宣传视觉。
   - `assets/brand/`：logo、角色/吉祥物、字体、图标和其他可复用品牌资产。
2. 在下面对应表格里添加文件名、场景和设计结论。
3. 标记案例状态：已确认、探索中、已过期或反例。

## 案例提炼清单

查看一个案例时，需要提炼：

- 它服务哪个用户角色和业务步骤。
- 它的视觉层级为什么有效。
- 哪些颜色、组件或布局模式可以复用。
- 哪些部分不应该脱离当前场景复用。
- 它是当前已确认案例、历史案例，还是只适合作为灵感。
